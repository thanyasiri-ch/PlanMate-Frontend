# PlanMate Frontend

Frontend application for PlanMate, a study companion platform that helps students prepare for exams through personalized planning, productivity tracking, and motivation-driven study tools.

## Features

- Study Planner
- Productivity Dashboard
- Focus Mode
- Peer Progress Tracking
- Real-time Notifications

## Tech Stack

- Vue.js
- TypeScript
- Tailwind CSS
- Firebase

## Run Locally

```bash
npm install
npm run dev
```
